

# Magic-PDF

便捷、准确的将PDF转换成Markdown文档


### 上手指南

###### 开发前的配置要求

python 3.9+

###### **安装步骤**

1.Clone the repo

```sh
git clone https://github.com/myhloli/Magic-PDF.git
```

2.Install the requirements

```sh
pip install -r requirements.txt
```

3.Run the main script

```sh
use demo/text_demo.py
or
use demo/ocr_demo.py
```
